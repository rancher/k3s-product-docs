K3sは、公式のhttps://www.cncf.io[CNCF] サンドボックスプロジェクトであり、リソースに制限のある遠隔地やIoTデバイス上での運用ワークロード用に設計された、軽量でありながら強力な認定Kubernetesディストリビューションを提供します。SUSE® Rancher Primeと併用すると、K3sはインストールが簡単で、軽量でありながら高可用性のKubernetesディストリビューションとなり、Rancherオーケストレーションプラットフォーム内で簡単に管理できます。

== 主な機能

* 単一のバイナリまたは最小限のコンテナイメージとして配布されます。
* デフォルトのストレージバックエンドとしてsqlite3に基づく軽量データストア。etcd3、MySQL、Postgresも利用可能です。
* TLSやオプションの複雑さを多く処理するシンプルなランチャーにラップされています。
* 軽量環境に対して合理的なデフォルト設定で、デフォルトで安全です。
* すべてのKubernetesコントロールプレーンコンポーネントの操作は、単一のバイナリとプロセスにカプセル化されており、K3sは証明書の配布などの複雑なクラスター操作を自動化および管理できます。
* 外部依存関係は最小限に抑えられており、唯一の要件は最新のカーネルとcgroupマウントです。
* 簡単な「バッテリー同梱」クラスター作成のために必要な依存関係をパッケージ化しています：
 ** containerd / cri-dockerdコンテナランタイム（CRI）
 ** Flannelコンテナネットワークインタフェース（CNI）
 ** CoreDNSクラスターDNS
 ** Traefik Ingressコントローラー
 ** ServiceLBロードバランサーコントローラー
 ** Kube-routerネットワークポリシーコントローラー
 ** Local-path-provisioner永続ボリュームコントローラー
 ** Spegel分散コンテナイメージレジストリミラー
 ** ホストユーティリティ（iptables、socatなど）
