= 関連プロジェクト
:page-languages: [en, ja, ko, zh]
:revdate: 2025-03-31
:page-revdate: {revdate}

K3sディストリビューションを実装するプロジェクトは、コミュニティを拡大するための歓迎すべき追加です。これらのプロジェクトは、以下のことに役立ちます：

* K3sの機能と潜在的なアプリケーションを探求する。
* 異なるプラットフォームや環境でK3sの機能を拡張する。
* 大規模な高可用性（HA）K3sクラスターを作成するプロセスを簡素化する。

== k3s-ansible

マルチノードK3sクラスターを起動し、ansibleに慣れているユーザーは、https://github.com/k3s-io/k3s-ansible[k3s-io/k3s-ansible]リポジトリを確認してください。このansibleプレイブックのセットは、ノードにK3sをインストールする便利な方法を提供し、インストールプロセスではなくクラスターの設定に集中できるようにします。

== k3sup

K3sクラスターのセットアッププロセスを簡素化する別のプロジェクトは、https://github.com/alexellis/k3sup[k3sup]です。このプロジェクトはgolangで書かれており、ノードへのSSHアクセスのみが必要です。また、組み込みのetcdだけでなく、外部データストアを使用してK3sをデプロイする便利な方法も提供します。

== autok3s

別のプロビジョニングツールであるhttps://github.com/cnrancher/autok3s[autok3s]は、さまざまなクラウドプロバイダー、VM、およびローカルマシンにわたってK3sクラスターをプロビジョニングするためのGUIを提供します。このツールは、K3sクラスターをプロビジョニングするためにグラフィカルインターフェースを好むユーザーにとって便利です。

== hetzner-k3s

Hetzner Cloudにデプロイするユーザーは、link:https://github.com/vitobotta/hetzner-k3s[hetzner-k3s]を確認してください。このCLIツールはCrystalで書かれており、Hetzner Cloud上にK3sクラスターをセットアップするために必要な追加の手順を処理します。
