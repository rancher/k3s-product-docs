= ネットワーキング
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

このセクションでは、K3sのネットワーク設定の手順が含まれています。

xref:networking/basic-network-options.adoc[基本ネットワークオプション]では、フランネルやシングル/デュアルスタック設定など、クラスターの基本的なネットワーク設定について説明します。

xref:networking/distributed-multicloud.adoc[ハイブリッド/マルチクラウド クラスター]では、リモートまたはハイブリッドノードにわたってK3sクラスターを展開するためのオプションについてのガイダンスを提供します。

xref:networking/multus-ipams.adoc[MultusおよびIPAMプラグイン]では、K3sでMultusを利用し、ポッドごとに複数のインターフェースを持つ方法についてのガイダンスを提供します。

xref:networking/networking-services.adoc[ネットワーキングサービス: DNS, ingress, etc]では、CoreDNS、Traefik、NetworkPolicyコントローラー、およびServiceLBコントローラーがk3s内でどのように機能するかを説明します。
