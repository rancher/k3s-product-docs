= インストール
:page-languages: [en, ja, ko, zh]
:revdate: 2025-09-11
:page-revdate: {revdate}

このセクションでは、さまざまな環境でのK3sのインストール手順について説明します。K3sのインストールを開始する前に、xref:installation/requirements.adoc[要件]を満たしていることを確認してください。

xref:installation/configuration.adoc[設定オプション]では、K3sをインストールする際に利用可能なオプションについて案内します。

xref:installation/private-registry.adoc[プライベートレジストリ設定]では、コンテナイメージレジストリの認証とミラーリングを設定するための`registries.yaml`の使用について説明します。

xref:installation/registry-mirror.adoc[埋め込みレジストリミラー]では、ノード間でのイメージのピアツーピア共有のために、埋め込まれた分散イメージレジストリミラーを有効にする方法を示します。

xref:installation/airgap.adoc[エアギャップインストール]では、インターネットに直接アクセスできない環境でK3sをセットアップする方法について詳しく説明します。

xref:installation/server-roles.adoc[サーバーロールの管理]では、専用の`control-plane`または`etcd`サーバーでK3sをセットアップする方法について詳しく説明します。

xref:installation/packaged-components.adoc[パッケージコンポーネントの管理]では、パッケージコンポーネントを無効にする方法、または自動デプロイされるマニフェストを使用して独自のものをインストールする方法について詳しく説明します。

xref:installation/uninstall.adoc[K3sのアンインストール]では、ホストからK3sを削除する方法について詳しく説明します。
