= K3sのアンインストール
:page-languages: [en, ja, ko, zh]
:revdate: 2026-01-05
:page-revdate: {revdate}

[CAUTION]
====
K3sのアンインストールはデータ損失を引き起こす可能性があります！
====

K3sをインストールする際にインストールスクリプトを使用した場合、アンインストール用のスクリプトがインストール中に生成されました。

アンインストールスクリプトを実行すると、K3sとすべての実行中のポッドが停止し、ローカルクラスタのデータストア、xref:add-ons/storage.adoc#_setting_up_the_local_storage_provider[Local Storage]の永続ボリュームデータ、ノード設定、およびすべてのスクリプトとCLIツールが削除されます。

外部データストアや、外部Kubernetes永続ボリュームを使用して作成されたポッドからのデータは削除されません。

アンインストールして再インストールした後にノードを既存のクラスターに再参加させる予定がある場合は、ノードパスワードシークレットが削除されるように、クラスターからノードを削除してください。詳細については、xref:architecture.adoc#_how_agent_node_registration_works[Node Registration]のドキュメントを参照してください。

== サーバーのアンインストール

サーバーノードからK3sをアンインストールするには、次のコマンドを実行します：

[,bash]
----
/usr/local/bin/k3s-uninstall.sh
----

== エージェントのアンインストール

エージェントノードからK3sをアンインストールするには、次のコマンドを実行します：

[,bash]
----
/usr/local/bin/k3s-agent-uninstall.sh
----
