= Instalação
:page-languages: [en, ja, ko, zh]
:revdate: 2025-09-11
:page-revdate: {revdate}

Esta seção contém instruções para instalar o K3s em vários ambientes. Por favor, certifique-se de que atendeu aos xref:installation/requirements.adoc[Requisitos] antes de começar a instalar o K3s.

xref:installation/configuration.adoc[Opções de Configuração] fornece orientações sobre as opções disponíveis para você ao instalar o K3s.

xref:installation/private-registry.adoc[Configuração de Registro Privado] abrange o uso de `registries.yaml` para configurar a autenticação e o espelhamento do registro de imagens de contêiner.

xref:installation/registry-mirror.adoc[Espelho de Registro Embutido] mostra como habilitar o espelho de registro de imagens distribuído embutido, para compartilhamento de imagens entre nós de forma peer-to-peer.

xref:installation/airgap.adoc[Instalação em Air Gap] detalha como configurar o K3s em ambientes que não têm acesso direto à Internet.

xref:installation/server-roles.adoc[Gerenciando Funções de Servidor] detalha como configurar o K3s com servidores dedicados `control-plane` ou `etcd`.

xref:installation/packaged-components.adoc[Gerenciando Componentes Empacotados] detalha como desabilitar componentes empacotados ou instalar os seus próprios usando manifests de auto-implantação.

xref:installation/uninstall.adoc[Desinstalando o K3s] detalha como remover o K3s de um host.
