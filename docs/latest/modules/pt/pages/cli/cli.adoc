= Ferramentas CLI
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

O binário K3s contém uma série de ferramentas adicionais que ajudam a gerenciar seu cluster.

|===
| Comando | Descrição

| `k3s server`
| Execute um nó de servidor K3s, que inicia os componentes Kubernetes `apiserver`, `scheduler`, `controller-manager` e `cloud-controller-manager`, além de um datastore e os componentes do agente. Consulte a documentação do comando xref:cli/server.adoc[`k3s server`] para obter mais informações.

| `k3s agent`
| Execute o nó do agente K3s, que inicia o controlador de política de rede `containerd`, `flannel`, `kube-router` e os componentes Kubernetes `kubelet` e `kube-proxy`. Consulte a documentação do comando xref:cli/agent.adoc[`k3s agent`] para obter mais informações.

| `k3s kubectl`
| Execute o comando embutido https://kubernetes.io/docs/reference/kubectl[`kubectl`]. Esta é uma CLI para interagir com o apiserver do Kubernetes.  Se a variável de ambiente `KUBECONFIG` não estiver definida, isso tentará automaticamente usar o kubeconfig em `/etc/rancher/k3s/k3s.yaml`.

| `k3s crictl`
| Execute o comando embutido https://github.com/kubernetes-sigs/cri-tools/blob/master/docs/crictl.md[`crictl`]. Esta é uma CLI para interagir com a interface de tempo de execução de contêiner do Kubernetes (CRI). Útil para depuração.

| `k3s ctr`
| Execute o comando embutido https://github.com/projectatomic/containerd/blob/master/docs/cli.md[`ctr`]. Esta é uma CLI para containerd, o daemon de contêiner usado pelo K3s. Útil para depuração.

| `k3s token`
| Gerenciar tokens de bootstrap. Consulte a documentação do comando xref:cli/token.adoc[`k3s token`] para obter mais informações.

| `k3s etcd-snapshot`
| Realize backups sob demanda dos dados do cluster K3s e faça upload para o S3. Consulte a documentação do comando xref:cli/etcd-snapshot.adoc[`k3s etcd-snapshot`] para obter mais informações.

| `k3s secrets-encrypt`
| Configure o K3s para criptografar segredos ao armazená-los no cluster. Consulte a documentação do comando xref:cli/secrets-encrypt.adoc[`k3s secrets-encrypt`] para obter mais informações.

| `k3s certificate`
| Gerenciar certificados do K3s. Consulte a documentação do comando xref:cli/certificate.adoc[`k3s certificate`] para obter mais informações.

| `k3s completion`
| Gerar scripts de conclusão de shell para k3s

| `k3s help`
| Mostra uma lista de comandos ou ajuda para um comando.
|===
