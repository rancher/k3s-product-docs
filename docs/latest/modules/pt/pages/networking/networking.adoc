Esta seção contém instruções para configurar a rede no K3s.

xref:networking/basic-network-options.adoc[Opções Básicas de Rede] abrange a configuração básica de rede do cluster, como flannel e configurações de pilha única/dupla

xref:networking/multus-ipams.adoc[Plugins Multus e IPAM] fornece orientações para aproveitar o Multus no K3s a fim de ter várias interfaces por pod

xref:networking/networking-services.adoc[Serviços de rede: DNS, ingress, etc] explica como o CoreDNS, o Traefik, o controlador de política de rede e o controlador ServiceLB funcionam dentro do k3s
