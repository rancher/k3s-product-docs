= Herramientas CLI
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

El binario de K3s contiene una serie de herramientas adicionales que te ayudan a gestionar tu clúster.

|===
| Comando | Descripción

| `k3s server`
| Ejecuta un nodo de servidor K3s, que lanza los componentes de Kubernetes `apiserver`, `scheduler`, `controller-manager` y `cloud-controller-manager`, además de un almacén de datos y los componentes del agente. Consulta la documentación del comando xref:cli/server.adoc[`k3s server` ] para más información.

| `k3s agent`
| Ejecuta el nodo agente de K3s, que lanza `containerd``flannel``kube-router` el controlador de directivas de red, y los componentes de Kubernetes `kubelet` y `kube-proxy`. Consulta la documentación del comando xref:cli/agent.adoc[`k3s agent` ] para más información.

| `k3s kubectl`
| Ejecuta el comando embebido https://kubernetes.io/docs/reference/kubectl[`kubectl` ]. Esta es una CLI para interactuar con el apiserver de Kubernetes.  Si la variable de entorno `KUBECONFIG` no está configurada, esto intentará automáticamente usar el kubeconfig en `/etc/rancher/k3s/k3s.yaml`.

| `k3s crictl`
| Ejecuta el comando embebido https://github.com/kubernetes-sigs/cri-tools/blob/master/docs/crictl.md[`crictl` ]. Esta es una CLI para interactuar con la interfaz de entorno de ejecución de contenedor (CRI) de Kubernetes. Útil para depuración.

| `k3s ctr`
| Ejecuta el comando embebido https://github.com/projectatomic/containerd/blob/master/docs/cli.md[`ctr` ]. Esta es una CLI para containerd, el daemon de contenedores utilizado por K3s. Útil para depuración.

| `k3s token`
| Gestiona los tokens de inicio. Consulta la documentación del comando xref:cli/token.adoc[`k3s token` ] para más información.

| `k3s etcd-snapshot`
| Realiza copias de seguridad bajo demanda de los datos del clúster K3s y súbelas a S3. Consulta la documentación del comando xref:cli/etcd-snapshot.adoc[`k3s etcd-snapshot` ] para más información.

| `k3s secrets-encrypt`
| Configura K3s para cifrar secretos al almacenarlos en el clúster. Consulta la documentación del comando xref:cli/secrets-encrypt.adoc[`k3s secrets-encrypt` ] para más información.

| `k3s certificate`
| Gestiona los certificados de K3s. Consulta la documentación del comando xref:cli/certificate.adoc[`k3s certificate` ] para más información.

| `k3s completion`
| Genera scripts de finalización de shell para k3s

| `k3s help`
| Muestra una lista de comandos o ayuda para un comando.
|===
