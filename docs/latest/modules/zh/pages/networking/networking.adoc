本节包含有关在 K3s 中配置网络的说明。

xref:networking/multus-ipams.adoc[Multus 和 IPAM 插件] 提供了在 K3s 中利用 Multus 的指导，以便每个 Pod 拥有多个接口。

xref:networking/networking-services.adoc[网络服务：DNS、ingress 等] 解释了 CoreDNS、Traefik、网络策略控制器和 ServiceLB 控制器在 k3s 中的工作原理。
