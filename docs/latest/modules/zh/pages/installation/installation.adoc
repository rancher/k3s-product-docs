= 安装
:page-languages: [en, ja, ko, zh]
:revdate: 2025-09-11
:page-revdate: {revdate}

本节包含在各种环境中安装 K3s 的说明。在开始安装 K3s 之前，请确保您已满足 xref:installation/requirements.adoc[要求]。

xref:installation/configuration.adoc[配置选项] 提供有关安装 K3s 时可用选项的指导。

xref:installation/private-registry.adoc[私有注册表配置] 涉及使用 `registries.yaml` 来配置容器镜像注册表的身份验证和镜像。

xref:installation/registry-mirror.adoc[嵌入式注册表镜像] 显示如何启用嵌入式分布式镜像注册表镜像，以便在节点之间进行点对点的镜像共享。

xref:installation/airgap.adoc[隔离安装] 详细说明如何在没有直接访问互联网的环境中设置 K3s。

xref:installation/server-roles.adoc[管理服务器角色] 详细说明如何使用专用的 `control-plane` 或 `etcd` 服务器设置 K3s。

xref:installation/packaged-components.adoc[管理软件包组件] 详细说明如何禁用软件包组件，或使用自动部署清单安装您自己的组件。

xref:installation/uninstall.adoc[卸载 K3s] 详细说明如何从主机中去除 K3s。
